# Modeling Water Distribution in the Water Cycle
CMSE 201 Final Project  
Author: Yihan Wang

## Project Overview
This project builds a computational compartmental model to simulate how water moves through different carriers in the water cycle, such as oceans, atmosphere, groundwater, and glaciers.

The goal is to investigate what factors influence the percentage of total water stored in each part of the water cycle.

## Tools Used
- Python
- NumPy
- Matplotlib
- Jupyter Notebook

## Project File
The full analysis and model implementation can be found in:

`Project_water_cycle.ipynb`

## Research Question
What affects the percentage of total water in different carriers in the water cycle?
